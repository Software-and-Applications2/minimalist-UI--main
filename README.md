<h3>How Authentication Works</h3>

<div>
  <pre style="background: #1e1e2e; color: #cdd6f4; padding: 16px; border-radius: 10px; border-left: 4px solid #7c3aed; font-size: 14px;">
<span style="color: #6c7086;">/* Authentication Flow */</span>

<span style="color: #89b4fa;">1️⃣</span> Admin navigates to <span style="color: #f9e2af;">/admin/login</span>
<span style="color: #89b4fa;">2️⃣</span> Enters password → form calls <span style="color: #f9e2af;">POST /api/auth</span>
<span style="color: #89b4fa;">3️⃣</span> Netlify Function compares against <span style="color: #f9e2af;">ADMIN_PASSWORD</span> env var
<span style="color: #89b4fa;">4️⃣</span> On success → token stored in <span style="color: #f9e2af;">localStorage</span>
<span style="color: #89b4fa;">5️⃣</span> Redirected to <span style="color: #f9e2af;">/admin</span> dashboard
<span style="color: #89b4fa;">6️⃣</span> All <span style="color: #f9e2af;">/admin/*</span> routes protected by <span style="color: #38bdf8;">ProtectedRoute</span> component</pre>
</div>
